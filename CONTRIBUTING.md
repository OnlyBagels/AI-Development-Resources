# Contributing

Contributions are welcome.

Good contributions include:

- Better prompt recipes.
- Better Claude Code agent definitions.
- Safer settings examples.
- New checklists.
- Real-world examples with private details removed.
- Corrections when Claude Code behavior changes.

## Style

- Be practical.
- Prefer examples over theory.
- Keep docs concise.
- Avoid hype.
- Include non-goals and failure modes.

## Pull request checklist

- [ ] I tested or manually reviewed the change.
- [ ] I kept examples Claude-focused.
- [ ] I did not include secrets or private project data.
- [ ] I updated related docs if needed.
