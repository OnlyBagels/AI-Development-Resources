# Checklist: Before Prompting

- [ ] Is this one task?
- [ ] Did I state the desired output format?
- [ ] Did I include only relevant context?
- [ ] Did I list constraints and non-goals?
- [ ] Did I say how success should be validated?
- [ ] Did I choose the cheapest reliable model?
- [ ] Should this use a Haiku worker instead of the main model?
- [ ] Should this be a new chat?
